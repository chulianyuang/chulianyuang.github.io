今天是个很特别的日子，看了有一会视频了，今天就开始上传第一篇关于 Java 入门的博文，其实在我看来学了 C 语言之后再来学 Java 感觉真的不要太简单。不过从小到大家人们都对我说不要眼高手低，所以我还是系统的从头到尾学一遍，当复习了。

今天给大家带来的是斯坦福大学的入门级练习题：

# 约会

您和您的约会对象在餐厅里面正在约会。

键盘录入两个整数，表示你和你约会对象衣服的时髦度。（手动录入 0~10 之间的整数，不能录入其他）

如果你的时髦程度大于你对象的时髦程度，相亲就成功，输出 true。

否则输出 false.

加油～
