  2. 2. 第二章 Java 编程基础
    1. 2.1. 请编写程序，实现计算 “1+3+5+7+・・・+99” 的值。要求如下：
      1. 2.1.1. 1. 使用循环语句实现自然数 1~99 的遍历。
      2. 2.1.2. 2. 在遍历过程中，通过条件判断当前遍历的数是否为奇数，如果是就累加，否则不加。
    2. 2.2. 请使用冒泡排序算法编写程序，实现对数组 {25，24，12，76，101，96，28} 的排序
  3. 3. 第三章 面向对象（上）
    1. 3.1. 请按照以下要求设计一个学生类 Student, 并进行调试。要求如下：
      1. 3.1.1. Student 类中包含姓名，成绩两个属性。
      2. 3.1.2. 分别给这两个属性定义两个方法，一个方法用于设置值，另一个方法用于获取值。
      3. 3.1.3. Student 类中定义一个无参的构造方法和一个接收两个参数的构造方法，两个参数分别为姓名和属性赋值。
      4. 3.1.4. 在测试类中创建两个 Student 对象，一个使用无参的构造方法，然后调用方法给姓名和成绩赋值，另一个使用有参的构造方法，在构造方法中给姓名和成绩赋值。
    2. 3.2. 定义一个这样的数列：0，1，1，2，3，5，・・・（斐波那契数列）。要求如下：
      1. 3.2.1. 使用递归方法获取第 n 个数的数值。
  4. 4. 第四章 面向对象（下）
    1. 4.1. 设计一个学生类 Student 和它的一个子类 Undergraduate。要求如下：
      1. 4.1.1. Student 类有 name 和 age 属性，一个包含两个参数的构造方法，用于给 name 和 age 属性赋值，一个 show () 方法打印 Student 的属性信息。
      2. 4.1.2. 本科生类 Undergraduate 增加一个 degree（学位）属性。有一个包含三个参数的构造方法，前两个参数用于给继承的 name 和 age 属性赋值，第三个参数给 degree 专业赋值，一个 show () 方法用于打印 Undergraduate 的属性信息。
      3. 4.1.3. 在测试类中分别创建 Student 对象和 Undergraduate 对象，调用他们的 show ().
    2. 4.2. 设计一个 Shape 接口和它的两个实现类 Square 和 Circle。要求如下：
      1. 4.2.1. Shape 接口中有一个抽象方法 area (), 方法接收有一个 double 类型的参数，返回一个 double 类型的结果。
      2. 4.2.2. Square 和 Circle 中实现了 Shape 接口的 area () 抽象方法，分别求正方形和圆形的面积并返回。
      3. 4.2.3. 在测试类中创建 Square 和 Circle 对象，计算边长为 2 的正方形面积和半径为 3 的圆形面积。
