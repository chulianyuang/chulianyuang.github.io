# 字符串转数组

# split () 方法

使用 split() 方法，该方法可以根据指定的分隔符将字符串分割为子字符串，并返回一个新的数组。例如， "abc".split("") 会返回 ["a", "b", "c"]

# ES6 的展开运算符…

使用 ES6 的展开运算符 ... ，该运算符可以将一个可迭代对象（如字符串）展开为多个元素，并放在一个数组字面量中。例如， [..."abc"] 会返回 ["a", "b", "c"]

# Array.from () 方法

使用 Array.from() 方法，该方法可以从一个类似数组或可迭代对象创建一个新的数组实例。例如， Array.from("abc") 会返回 ["a", "b", "c"]

# Object.assign () 方法

使用 Object.assign() 方法，该方法可以将一个或多个源对象的可枚举属性复制到目标对象中。例如， Object.assign([], "abc") 会返回 ["a", "b", "c"] 。

# toCharArray () 方法

使用 toCharArray() 方法，该方法可以将字符串转换为字符数组。例如， "abc".toCharArray() 会返回 ['a', 'b', 'c'] 。
